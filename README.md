# 📊 Personal Finance Tracker (Fullstack Diploma Project)

Сучасний веб-застосунок для обліку особистих фінансів, створений у рамках дипломної роботи. Дозволяє користувачам додавати доходи та витрати, керувати категоріями, встановлювати ліміти, отримувати математичний прогноз витрат на наступний місяць за допомогою лінійної регресії, а також візуалізувати динаміку балансу.

## 🚀 Технологічний стек

**Frontend:**
- **Core:** React (TypeScript) + Vite
- **Styling:** Tailwind CSS (v4)
- **Routing:** React Router DOM (із Protected Routes)
- **State Management:** React Query (@tanstack/react-query)
- **Data Viz:** Chart.js + react-chartjs-2 (Круговий та Лінійний графіки)
- **Network:** Axios (з interceptors для обробки 401 Unauthorized)
- **UI/UX:** React Hot Toast (плавні сповіщення)

**Backend:**
- **Core:** Node.js + Express (TypeScript)
- **Database:** PostgreSQL
- **ORM:** Prisma (зі схемою та міграціями)
- **Validation:** Zod (сувора декларативна валідація даних)
- **Security:** JWT + bcryptjs (аутентифікація), Helmet, Express Rate Limit
- **Logging:** Pino + pino-pretty (професійне логування)
- **Testing:** Jest + ts-jest (Unit-тестування алгоритмів)

**DevOps:**
- Docker & Docker Compose

## 🛠️ Ключовий функціонал
- **Аутентифікація**: Безпечна реєстрація та логін (JWT). Автоматичний вихід при простроченні токена.
- **Керування даними (CRUD)**: Створення, читання, видалення транзакцій та категорій. Вибір кастомної дати транзакції.
- **Категорії та ліміти**: Встановлення місячних лімітів на витрати з автоматичним Toast-сповіщенням при їх перевищенні.
- **Аналітика та Прогнозування**: Розрахунок прогнозу витрат на наступний місяць на базі алгоритму лінійної регресії (математична логіка покрита unit-тестами).
- **Візуалізація**: Інтерактивні графіки структури витрат (Doughnut) та накопичувального балансу по місяцях (Line chart).
- **Пагінація та Фільтрація**: Backend API підтримує отримання транзакцій з пагінацією та фільтрами за датою/типом.
- **Експорт**: Завантаження історії транзакцій у форматі CSV (з підтримкою кирилиці через BOM).

## 🐳 Запуск за допомогою Docker (Рекомендовано)

Найшвидший спосіб розгорнути проєкт з базою даних, сервером та клієнтом однією командою.

1. Клонуйте репозиторій:
   ```bash
   git clone <url-вашого-репозиторію>
   cd finance-diploma
   ```

2. Запустіть контейнери:
   ```bash
   docker-compose up --build
   ```

**Після запуску:**
* Frontend буде доступний за адресою: `http://localhost:5173`
* Backend API: `http://localhost:5000/api`
* База даних PostgreSQL автоматично ініціалізується та виконає міграції.

## ⚙️ Встановлення та запуск локально (Без Docker)

### 1. Налаштування Backend
```bash
cd server
npm install
```
- Створіть файл `.env` на основі `.env.example` та вкажіть ваш `DATABASE_URL` і `JWT_SECRET`.
- Застосуйте міграції та запустіть сервер:
```bash
npx prisma migrate dev
npm run dev
```

### 2. Налаштування Frontend
```bash
cd client
npm install
```
- Створіть файл `.env` на основі `.env.example` та перевірте `VITE_API_URL`.
- Запустіть клієнтську частину:
```bash
npm run dev
```

## 🛡️ Безпека та Архітектура
Проєкт розроблено з урахуванням сучасних стандартів безпеки. Всі вхідні дані проходять сувору типізацію та валідацію через бібліотеку **Zod**. Для захисту від DDoS атак та XSS вразливостей використовуються **express-rate-limit** та **helmet**. Помилки сервера обробляються та виводяться через професійний логер **Pino**, а неконтрольований доступ на клієнті блокується за допомогою **Protected Routes**.
